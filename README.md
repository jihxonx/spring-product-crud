# Product CRUD Practice

Spring Boot와 JPA를 사용하여  
**상품(Product) 도메인의 CRUD 기능을 구현한 연습 프로젝트**입니다.

Spring의 기본 구조(Controller / Service / Repository)와  
의존성 주입(DI), JPA 사용 흐름을 익히는 것을 목표로 합니다.

---

## 🛠 Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA 
- H2 Database
- Gradle
- Postman (API 테스트)
